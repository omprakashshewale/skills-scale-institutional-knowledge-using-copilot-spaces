# OctoAcme Project Management Docs — README

This folder contains OctoAcme's project management process documents. This README provides a brief overview of the project management approach and an index with links to each process document.

## Brief overview of OctoAcme project management processes

OctoAcme runs projects with an emphasis on customer value, iterative delivery, clear ownership, and continuous improvement.

- Customer-first and data-informed decision making.
- Iterative delivery: break work into small, testable increments and iterate on results.
- Clear ownership: each project has a named Project Manager (PM) and Product Lead (PdM).
- Defined lifecycle stages: Initiation → Planning → Execution → Release → Retrospective.
- Strong communication cadence: daily standups, weekly delivery syncs, demos at milestones, and stakeholder updates.
- Risk identification and escalation paths are documented and monitored regularly.
- Continuous improvement: capture retrospective action items and measure impact.
